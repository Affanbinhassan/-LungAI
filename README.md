# 🫁 LungAI

> **AI-Powered Lung Cancer Prediction System using Stacking Ensemble Learning**

LungAI is an intelligent healthcare application that predicts the likelihood of lung cancer using machine learning techniques. The project combines multiple ensemble learning algorithms with a Flask-based web application to provide an interactive and user-friendly prediction system.

---

## 📖 Overview

Early detection of lung cancer can significantly improve patient outcomes. LungAI leverages a Stacking Ensemble Machine Learning model trained on clinical and lifestyle data to provide accurate risk predictions through an easy-to-use web interface.

The project demonstrates the integration of Artificial Intelligence, Data Science, and Web Development into a real-world healthcare application.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Affanbinhassan/LungAI.git
```

Move into the project folder

```bash
cd LungAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
